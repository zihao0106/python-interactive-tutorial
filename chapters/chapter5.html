📚 列表基础

列表（List）是Python中最常用的数据类型之一，用来存储多个数据项。列表是有序的、可变的，可以包含不同类型的数据。

创建列表

💻 创建列表 
# 创建空列表
empty_list = []
print("空列表:", empty_list)

# 创建有元素的列表
fruits = ["苹果", "香蕉", "橙子"]
print("水果列表:", fruits)

# 包含不同类型数据的列表
mixed_list = [1, "hello", 3.14, True]
print("混合列表:", mixed_list)

# 使用list()函数创建列表
numbers = list(range(1, 6))
print("数字列表:", numbers)

🔧 列表操作

访问列表元素

💻 访问列表元素 
fruits = ["苹果", "香蕉", "橙子", "葡萄", "西瓜"]

# 正向索引（从0开始）
print("第一个水果:", fruits[0])
print("第三个水果:", fruits[2])

# 负向索引（从-1开始）
print("最后一个水果:", fruits[-1])
print("倒数第二个:", fruits[-2])

# 列表切片
print("前三个水果:", fruits[:3])
print("后两个水果:", fruits[-2:])
print("中间的水果:", fruits[1:4])

修改列表

💻 修改列表 
fruits = ["苹果", "香蕉", "橙子"]
print("原始列表:", fruits)

# 修改单个元素
fruits[1] = "芒果"
print("修改后:", fruits)

# 修改多个元素
fruits[0:2] = ["草莓", "樱桃"]
print("批量修改:", fruits)

📋 列表方法

添加元素

💻 添加元素 
fruits = ["苹果", "香蕉"]
print("初始列表:", fruits)

# append() - 在末尾添加一个元素
fruits.append("橙子")
print("添加橙子:", fruits)

# insert() - 在指定位置插入元素
fruits.insert(1, "草莓")
print("插入草莓:", fruits)

# extend() - 添加多个元素
fruits.extend(["葡萄", "西瓜"])
print("扩展列表:", fruits)

删除元素

💻 删除元素 
fruits = ["苹果", "香蕉", "橙子", "草莓", "葡萄"]
print("原始列表:", fruits)

# remove() - 删除指定值的第一个元素
fruits.remove("香蕉")
print("删除香蕉:", fruits)

# pop() - 删除并返回指定位置的元素（默认最后一个）
last_fruit = fruits.pop()
print("弹出的水果:", last_fruit)
print("弹出后列表:", fruits)

# del - 删除指定位置的元素
del fruits[0]
print("删除第一个:", fruits)

其他常用方法

💻 其他列表方法 
numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5]
print("原始列表:", numbers)

# count() - 统计元素出现次数
print("1出现的次数:", numbers.count(1))

# index() - 查找元素的索引
print("4的索引:", numbers.index(4))

# sort() - 排序（修改原列表）
numbers.sort()
print("升序排列:", numbers)

# reverse() - 反转列表
numbers.reverse()
print("反转列表:", numbers)

# len() - 获取列表长度
print("列表长度:", len(numbers))

📦 元组（Tuple）

元组是有序的、不可变的数据类型。一旦创建，就不能修改其内容。

创建和使用元组

💻 元组操作 
# 创建元组
point = (3, 4)
print("坐标点:", point)

# 创建单个元素的元组（注意逗号）
single_tuple = (42,)
print("单元素元组:", single_tuple)

# 不使用括号也可以创建元组
colors = "红", "绿", "蓝"
print("颜色元组:", colors)

# 访问元组元素
print("x坐标:", point[0])
print("y坐标:", point[1])

# 元组解包
x, y = point
print(f"解包后: x={x}, y={y}")

# 元组的长度和其他操作
print("元组长度:", len(point))
print("元组中的最大值:", max(point))
为什么使用元组？
  • 不可变性：数据安全，不会被意外修改
  • 性能：比列表更快，占用内存更少
  • 字典键：可以作为字典的键（列表不行）
  • 函数返回：适合返回多个值

🚀 列表推导式

列表推导式是Python的一个强大特性，可以用简洁的语法创建列表。

💻 列表推导式 
# 基本语法
squares = [x**2 for x in range(1, 6)]
print("平方数:", squares)

# 带条件的列表推导式
even_squares = [x**2 for x in range(1, 11) if x % 2 == 0]
print("偶数的平方:", even_squares)

# 处理字符串列表
words = ["hello", "world", "python", "programming"]
uppercase = [word.upper() for word in words]
print("大写单词:", uppercase)

# 条件表达式
numbers = [1, -2, 3, -4, 5]
abs_numbers = [x if x >= 0 else -x for x in numbers]
print("绝对值:", abs_numbers)

# 嵌套列表推导式
matrix = [[i*j for j in range(1, 4)] for i in range(1, 4)]
print("乘法表:")
for row in matrix:
    print(row)

💻 实践练习

练习1：购物清单管理

💻 购物清单管理 
# 创建购物清单
shopping_list = []

# 添加商品
shopping_list.extend(["牛奶", "面包", "鸡蛋", "苹果"])
print("购物清单:", shopping_list)

# 检查是否需要买牛奶
if "牛奶" in shopping_list:
    print("需要买牛奶")

# 买完面包，从清单中删除
shopping_list.remove("面包")
print("买完面包后:", shopping_list)

# 添加新商品
shopping_list.insert(0, "香蕉")  # 插入到开头
print("添加香蕉:", shopping_list)

# 清单排序
shopping_list.sort()
print("排序后:", shopping_list)

练习2：成绩处理

💻 成绩处理 
# 学生成绩
scores = [85, 92, 78, 96, 88, 76, 94, 82]
print("原始成绩:", scores)

# 计算统计信息
print("最高分:", max(scores))
print("最低分:", min(scores))
print("平均分:", sum(scores) / len(scores))

# 筛选优秀成绩（>=90分）
excellent = [score for score in scores if score >= 90]
print("优秀成绩:", excellent)

# 给所有成绩加5分奖励分
bonus_scores = [score + 5 for score in scores]
print("加奖励分后:", bonus_scores)

# 统计各分数段人数
above_90 = len([s for s in scores if s >= 90])
between_80_90 = len([s for s in scores if 80 <= s < 90])
below_80 = len([s for s in scores if s < 80])

print(f"90分以上: {above_90}人")
print(f"80-90分: {between_80_90}人")
print(f"80分以下: {below_80}人")
📋 本章总结
  • 列表是可变的有序序列，支持增删改查操作
  • 常用列表方法：append()、insert()、remove()、pop()、sort()等
  • 元组是不可变的有序序列，创建后不能修改
  • 列表推导式提供了创建列表的简洁语法
  • 根据数据特点选择使用列表或元组
  • 掌握索引、切片等序列通用操作
