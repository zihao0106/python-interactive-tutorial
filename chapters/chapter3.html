🧮 第3章：运算符与表达式

学习Python中的各种运算符，让计算机帮我们进行数学运算和逻辑判断。

🎯 学习目标
  • 掌握算术运算符的使用
  • 学会比较运算符和逻辑运算符
  • 理解运算符的优先级
  • 学会使用赋值运算符
  • 了解表达式的概念和计算

➕ 算术运算符

算术运算符用于执行基本的数学运算：

运算符 名称 示例 结果
+加法5 + 38
-减法5 - 32
*乘法5 * 315
/除法5 / 22.5
//整除5 // 22
%取余5 % 21
**幂运算2 ** 38
💻 算术运算示例 
# 算术运算符示例
a = 10
b = 3

print("加法:", a + b)      # 13
print("减法:", a - b)      # 7
print("乘法:", a * b)      # 30
print("除法:", a / b)      # 3.333...
print("整除:", a // b)     # 3
print("取余:", a % b)      # 1
print("幂运算:", a ** b)   # 1000

# 实际应用：计算圆的面积
import math
radius = 5
area = math.pi * radius ** 2
print(f"半径为{radius}的圆的面积: {area:.2f}")

⚖️ 比较运算符

比较运算符用于比较两个值，返回True或False：

运算符 名称 示例 结果
==等于5 == 5True
!=不等于5 != 3True
>大于5 > 3True
<小于5 < 3False
>=大于等于5 >= 5True
<=小于等于5 <= 3False
💻 比较运算示例 
# 比较运算符示例
score1 = 85
score2 = 92

print("score1 == score2:", score1 == score2)  # False
print("score1 != score2:", score1 != score2)  # True
print("score1 > 80:", score1 > 80)           # True
print("score1 < score2:", score1 < score2)   # True
print("score2 >= 90:", score2 >= 90)         # True

# 字符串也可以比较
name1 = "Alice"
name2 = "Bob"
print("name1 < name2:", name1 < name2)       # True (按字母顺序)

# 实际应用：判断成绩等级
score = 88
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"
print(f"分数{score}对应等级: {grade}")

🧠 逻辑运算符

逻辑运算符用于组合多个条件：

运算符 名称 说明 示例
and与两个条件都为真时结果为真True and False → False
or或至少一个条件为真时结果为真True or False → True
not非条件取反not True → False
💻 逻辑运算示例 
# 逻辑运算符示例
age = 20
has_license = True
score = 85

# and 运算符
can_drive = age >= 18 and has_license
print("可以开车:", can_drive)  # True

# or 运算符
is_excellent = score >= 90 or score >= 85
print("成绩优秀:", is_excellent)  # True

# not 运算符
is_minor = not (age >= 18)
print("是未成年人:", is_minor)  # False

# 复杂条件组合
weather = "sunny"
temperature = 25
is_good_day = weather == "sunny" and temperature > 20 and temperature < 30
print("今天天气不错:", is_good_day)

# 实际应用：用户权限检查
is_admin = False
is_owner = True
can_delete = is_admin or is_owner
print("可以删除文件:", can_delete)

📝 赋值运算符

赋值运算符用于给变量赋值，包括复合赋值运算符：

运算符 示例 等价于 说明
=x = 5x = 5简单赋值
+=x += 3x = x + 3加法赋值
-=x -= 3x = x - 3减法赋值
*=x *= 3x = x * 3乘法赋值
/=x /= 3x = x / 3除法赋值
💻 赋值运算示例 
# 赋值运算符示例
score = 80
print("初始分数:", score)

# 复合赋值运算符
score += 10  # 等价于 score = score + 10
print("加10分后:", score)

score -= 5   # 等价于 score = score - 5
print("减5分后:", score)

score *= 2   # 等价于 score = score * 2
print("翻倍后:", score)

score /= 3   # 等价于 score = score / 3
print("除以3后:", score)

# 实际应用：计数器
counter = 0
counter += 1  # 计数加1
counter += 1
counter += 1
print("计数器值:", counter)

# 实际应用：累加器
total = 0
numbers = [10, 20, 30, 40]
for num in numbers:
    total += num  # 累加
print("总和:", total)

🏆 运算符优先级

当表达式中有多个运算符时，Python按照优先级顺序计算：

优先级顺序（从高到低）：
  1. 括号：( )
  2. 幂运算：**
  3. 乘除：*, /, //, %
  4. 加减：+, -
  5. 比较：==, !=, <, >, <=, >=
  6. 逻辑非：not
  7. 逻辑与：and
  8. 逻辑或：or
💻 运算符优先级示例 
# 运算符优先级示例
result1 = 2 + 3 * 4        # 先乘后加：2 + 12 = 14
result2 = (2 + 3) * 4      # 括号优先：5 * 4 = 20
result3 = 2 ** 3 * 4       # 先幂运算：8 * 4 = 32
result4 = 10 / 2 * 3       # 从左到右：5 * 3 = 15

print("2 + 3 * 4 =", result1)
print("(2 + 3) * 4 =", result2)
print("2 ** 3 * 4 =", result3)
print("10 / 2 * 3 =", result4)

# 复杂表达式
x = 5
y = 3
z = 2
complex_result = x > y and y * z > 5 or x ** 2 < 30
print("复杂表达式结果:", complex_result)

# 建议：使用括号让表达式更清晰
clear_result = (x > y) and ((y * z) > 5) or ((x ** 2) < 30)
print("清晰表达式结果:", clear_result)

💪 实践练习

练习1：简单计算器

创建一个简单计算器，可以进行基本的数学运算：

💻 练习代码 
# 简单计算器
print("=== 简单计算器 ===")

# 获取两个数字
num1 = float(input("请输入第一个数字: "))
num2 = float(input("请输入第二个数字: "))

# 进行各种运算
print(f"\n{num1} 和 {num2} 的运算结果:")
print(f"加法: {num1} + {num2} = {num1 + num2}")
print(f"减法: {num1} - {num2} = {num1 - num2}")
print(f"乘法: {num1} * {num2} = {num1 * num2}")

# 避免除零错误
if num2 != 0:
    print(f"除法: {num1} / {num2} = {num1 / num2:.2f}")
    print(f"整除: {num1} // {num2} = {num1 // num2}")
    print(f"取余: {num1} % {num2} = {num1 % num2}")
else:
    print("除法: 不能除以零")

print(f"幂运算: {num1} ** {num2} = {num1 ** num2}")

# 比较运算
print(f"\n比较运算:")
print(f"{num1} > {num2}: {num1 > num2}")
print(f"{num1} == {num2}: {num1 == num2}")
print(f"{num1} != {num2}: {num1 != num2}")

# 逻辑运算示例
both_positive = num1 > 0 and num2 > 0
either_positive = num1 > 0 or num2 > 0
print(f"\n两个数都是正数: {both_positive}")
print(f"至少一个是正数: {either_positive}")
📋 本章总结
  • 算术运算符：+, -, *, /, //, %, ** 用于数学计算
  • 比较运算符：==, !=, >, <, >=, <= 用于比较值
  • 逻辑运算符：and, or, not 用于组合条件
  • 赋值运算符：=, +=, -=, *=, /= 用于给变量赋值
  • 运算符有优先级，使用括号可以改变计算顺序
  • 表达式是由变量、常量和运算符组成的计算式
