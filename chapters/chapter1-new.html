🌟 什么是Python？

Python是一种简单易学、功能强大的编程语言，由Guido van Rossum于1991年首次发布。

✨ Python的特点
  • 简洁明了：语法简单，易于阅读和编写
  • 跨平台：可在Windows、Mac、Linux上运行
  • 功能强大：拥有丰富的标准库和第三方库
  • 开源免费：完全免费使用
🎯 Python的应用领域
  • 网站开发
  • 数据分析
  • 人工智能
  • 自动化脚本
  • 科学计算

👋 第一个Python程序

让我们从经典的"Hello, World!"程序开始Python编程之旅：

# 这是我的第一个Python程序
print("Hello, World!")
print("欢迎来到Python编程世界！")
💡 代码解释：
  • # 开头的是注释，不会被执行
  • print() 函数用来输出信息到屏幕
  • 字符串需要用引号包围

📢 print()函数的使用

print()函数是Python中最常用的函数之一，用于在屏幕上显示信息：

# 输出文本
print("Hello Python!")

# 输出数字
print(42)
print(3.14)

# 输出多个内容
print("我的年龄是", 25, "岁")

# 使用分隔符
print("苹果", "香蕉", "橙子", sep=" - ")

# 控制结尾字符
print("第一行", end=" ")
print("第二行")
🔍 观察输出：
  • print()可以输出文本、数字等不同类型的数据
  • 用逗号分隔多个值，默认用空格连接
  • sep参数可以改变分隔符
  • end参数可以改变结尾字符

📝 代码注释

注释是程序员之间交流的重要工具，也是让代码更易理解的关键：

# 这是单行注释

"""
这是多行注释
可以写很多行
用来解释复杂的代码逻辑
"""

# 计算圆的面积
radius = 5
pi = 3.14159
area = pi * radius * radius  # 面积 = π × r²
print(f"半径为 {radius} 的圆的面积是 {area}")
💭 注释的好习惯：
  • 用 # 写单行注释
  • 用 """ 或 ''' 写多行注释
  • 注释要简洁明了，解释代码的目的
  • 复杂算法一定要加注释
