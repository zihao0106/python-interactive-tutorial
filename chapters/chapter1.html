🐍 什么是Python？

Python是一种简单易学、功能强大的编程语言。它有以下特点：

  • 简洁明了：代码易读易写
  • 功能强大：可以做网站、数据分析、人工智能等
  • 跨平台：Windows、Mac、Linux都能运行
  • 开源免费：任何人都可以使用

👋 第一个Python程序

让我们从最经典的"Hello, World!"开始：

💻 代码示例 
# 这是我的第一个Python程序
print("Hello, World!")
print("欢迎来到Python编程世界！")
💡 知识点：
  • print()函数用于在屏幕上显示文字
  • 文字需要用引号包围（单引号或双引号都可以）
  • 以#开头的行是注释，不会被执行

📝 代码注释

注释是给人看的说明文字，Python会忽略它们。注释非常重要，能帮助你和其他人理解代码。

💻 注释示例 
# 单行注释：计算两个数的和
result = 5 + 3  # 这也是注释

"""
多行注释：
这里可以写很多行的说明
通常用于函数或类的说明
"""

print("计算结果:", result)

📏 缩进规则

Python使用缩进来表示代码块，这是Python的一个重要特色：

💻 缩进示例 
# 正确的缩进示例
age = 18
if age >= 18:
    print("你已成年")  # 这行需要缩进
    print("可以投票")  # 这行也需要缩进
print("程序结束")      # 这行不需要缩进
⚠️ 注意：
  • Python中缩进通常使用4个空格
  • 同一代码块的缩进必须保持一致
  • 错误的缩进会导致程序报错
