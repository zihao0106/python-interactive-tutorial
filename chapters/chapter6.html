📚 字典基础

字典（Dictionary）是一种键值对（key-value）的数据结构，用大括号{}表示。字典是无序的、可变的，键必须是唯一的。

创建字典

💻 创建字典 
# 创建空字典
empty_dict = {}
print("空字典:", empty_dict)

# 创建有数据的字典
student = {
    "name": "张三",
    "age": 20,
    "major": "计算机科学"
}
print("学生信息:", student)
� 本章总结
  • 字典是键值对结构，支持快速查找和修改
  • 字典的键必须是不可变类型，值可以是任意类型
  • 常用字典方法：get()、keys()、values()、items()、update()
  • 集合用于存储唯一元素，支持数学集合运算
  • 集合运算：并集(|)、交集(&)、差集(-)、对称差集(^)
  • 字典和集合推导式提供简洁的创建语法
