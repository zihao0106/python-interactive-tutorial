📦 第2章：变量与数据类型

学习如何在Python中存储和使用不同类型的数据。

🎯 学习目标
  • 理解什么是变量，如何创建和使用变量
  • 掌握Python的基本数据类型
  • 学会数据类型转换
  • 了解变量命名规则
  • 学会使用input()函数获取用户输入

📊 什么是变量？

变量就像一个盒子，用来存储数据。我们可以给这个盒子起一个名字，以后就能通过这个名字来使用里面的数据。

💻 变量示例 
# 创建变量
name = "小明"
age = 18
height = 175.5

# 使用变量
print("姓名:", name)
print("年龄:", age)
print("身高:", height, "厘米")
💡 知识点：
  • 用 = 号给变量赋值
  • 变量名在左边，值在右边
  • Python会自动识别数据类型

🏷️ Python的基本数据类型

Python有几种基本的数据类型：

数字类型
  • 整数(int)：123, -456
  • 浮点数(float)：3.14, -2.5
其他类型
  • 字符串(str)："hello", '世界'
  • 布尔值(bool)：True, False
💻 数据类型示例 
# 不同的数据类型
整数 = 42
浮点数 = 3.14159
字符串 = "Hello, Python!"
布尔值 = True

# 查看数据类型
print("整数:", 整数, "类型:", type(整数))
print("浮点数:", 浮点数, "类型:", type(浮点数))
print("字符串:", 字符串, "类型:", type(字符串))
print("布尔值:", 布尔值, "类型:", type(布尔值))

📝 变量命名规则

给变量起名字要遵循一些规则：

✅ 正确的命名
  • name, age, student_count
  • userName, firstName
  • num1, score_2
  • 姓名, 年龄 (支持中文)
❌ 错误的命名
  • 2name (不能以数字开头)
  • my-age (不能有连字符)
  • class (不能用关键字)
  • my name (不能有空格)
💻 命名示例 
# 好的变量命名
student_name = "张三"
total_score = 95
is_passed = True

# 使用有意义的变量名
temperature_celsius = 25
user_email = "zhang@example.com"

print(f"学生: {student_name}")
print(f"总分: {total_score}")
print(f"是否通过: {is_passed}")
print(f"温度: {temperature_celsius}°C")

🔄 数据类型转换

有时我们需要将一种数据类型转换为另一种：

💻 类型转换示例 
# 类型转换示例
# 字符串转数字
age_str = "18"
age_int = int(age_str)
print("字符串:", age_str, "转换后:", age_int)

# 数字转字符串
score = 95
score_str = str(score)
print("数字:", score, "转换后:", score_str)

# 字符串转浮点数
height_str = "175.5"
height_float = float(height_str)
print("字符串:", height_str, "转换后:", height_float)

# 数字转布尔值
print("0转布尔值:", bool(0))
print("1转布尔值:", bool(1))

⌨️ 获取用户输入

使用input()函数可以让用户输入数据：

💻 用户输入示例 
# 注意：在网页环境中，input会使用模拟输入
name = input("请输入你的姓名: ")
print("你好,", name)

# 输入数字时需要转换类型
age_str = input("请输入你的年龄: ")
age = int(age_str)
print("你", age, "岁了")

# 更简洁的写法
height = float(input("请输入你的身高(米): "))
print(f"你的身高是 {height} 米")
⚠️ 注意：
  • input()函数返回的总是字符串
  • 如果需要数字，要用int()或float()转换
  • 在这个网页环境中，input会使用模拟输入

💪 实践练习

练习1：个人信息收集器

创建一个程序，收集并显示个人信息：

💻 练习代码 
# 个人信息收集器
print("=== 个人信息收集器 ===")

# 收集信息
name = input("姓名: ")
age = int(input("年龄: "))
height = float(input("身高(米): "))
is_student = input("是学生吗? (是/否): ") == "是"

# 计算一些有趣的信息
height_cm = height * 100
birth_year = 2024 - age

# 显示信息
print("\n=== 你的信息 ===")
print(f"姓名: {name}")
print(f"年龄: {age} 岁")
print(f"身高: {height} 米 ({height_cm} 厘米)")
print(f"大约出生年份: {birth_year}")
print(f"学生身份: {is_student}")

# 显示数据类型
print(f"\n姓名的类型: {type(name)}")
print(f"年龄的类型: {type(age)}")
print(f"身高的类型: {type(height)}")
print(f"学生身份的类型: {type(is_student)}")
📋 本章总结
  • 变量是存储数据的容器，用 = 赋值
  • Python有四种基本数据类型：int、float、str、bool
  • 变量命名要遵循规则：字母开头、不能有空格等
  • 使用 type() 查看数据类型
  • 可以用 int()、float()、str() 转换数据类型
  • input() 函数用于获取用户输入（返回字符串）
